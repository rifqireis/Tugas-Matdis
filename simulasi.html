# Node Score Visual
Cara membaca

Tips memakai sandbox.

01

Ukuran node = skor.

Setelah dijalankan, ukuran lingkaran sebanding dengan skor algoritma yang dipilih. Node terbesar = paling berpengaruh.

02

Coba preset Two Communities.

Lihat bagaimana Betweenness "menemukan" node jembatan — meski Degree-nya rendah, Betweenness-nya jauh lebih tinggi karena semua shortest path antar komunitas melewatinya.

03

Arah edge menentukan segalanya.

Coba preset Star — Hub punya banyak edge keluar (Hub->A), tapi PageRank-nya justru rendah karena tidak ada yang menunjuk ke Hub. Ubah ke Hub<->A dan lihat perbedaannya.
